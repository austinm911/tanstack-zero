# Tanstack React Router + Zero Starter

Created with:
`bunx create-tsrouter-app@latest my-app --template file-router --toolchain biome --add-ons shadcn,form`

Running the app

```bash
# Start the database
bun db:up

# Push the schema to the database
bun db:push

# Start Zero Cache
bun zero-cache

# Start dev server
bun dev 
```

## Stack

- Tanstack React Router
- Zero Sync
- Drizzle
- Shadcn
- Biome
